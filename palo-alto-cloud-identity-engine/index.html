Authentication


Authentication is the process of verifying the identity of a user. Palo Alto Cloud Identity Engine provides a variety of authentication methods to meet the needs of different organizations. These methods include:

  • Password-based authentication: This is the most common type of authentication. Users enter a username and password to access applications and resources.
  • Multi-factor authentication (MFA): MFA adds an extra layer of security by requiring users to provide multiple forms of identification. For example, a user might be required to enter a password and a one-time code sent to their mobile phone.
  • Single sign-on (SSO): SSO allows users to access multiple applications and resources using a single set of credentials. This makes it easier for users to access the applications and resources they need, and it also reduces the risk of password fatigue.

The authentication methods provided by Palo Alto Cloud Identity Engine are designed to be flexible and scalable. This allows organizations to choose the methods that best meet their security needs.
