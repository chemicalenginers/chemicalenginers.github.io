Curriculum


ASU's software engineering program features a rigorous curriculum that provides students with a deep understanding of the principles and practices of software development. This includes coursework in software development methodologies, software design, and software testing.

Software development methodologies provide a structured approach to software development, ensuring that projects are completed on time, within budget, and to the required quality standards. Students learn about different methodologies, such as Agile and Waterfall, and how to apply them effectively in real-world projects.

Software design is the process of creating a blueprint for a software system. Students learn about different design patterns and principles, and how to use them to create software that is maintainable, scalable, and efficient. Software testing is the process of evaluating software to ensure that it meets the specified requirements. Students learn about different testing techniques and tools, and how to use them to identify and fix defects.

The combination of coursework in software development methodologies, software design, and software testing provides students with a comprehensive understanding of the software development process. This knowledge is essential for success in the field of software engineering.
